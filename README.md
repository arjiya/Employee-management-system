# Employee Management System

## Description
A web application built with PHP and MySQL to manage employee records. Supports all CRUD operations: Create, Read, Update, Delete.

## Features
- Add, view, update, and delete employee records
- Responsive UI with HTML/CSS/JS
- Secure database integration with MySQL
- Data validation for accurate input

## Technologies
- PHP
- MySQL
- HTML / CSS / JavaScript

## Installation
1. Clone the repository: `git clone https://github.com/arjiya/Employee-management-system.git`
2. Create a MySQL database and import the provided SQL file.
3. Open `index.php` in your local server (XAMPP/WAMP).
